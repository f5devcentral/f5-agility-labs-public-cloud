HA Failover via Cloud Load Balancer (Optional)
================================================================================

In this module, you will:

- Disable Network Failover between the BIG-IPs to make the BIG-IPs Active-Active
- Create an AWS Network Load Balancer
- Test the LB-based failover by disabling virtual servers on the BIG-IPs to simulate failure events
- Validate the Network Load Balancer activity via the AWS console
