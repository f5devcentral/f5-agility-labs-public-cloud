F5 Analytics Integration with AWS --- TYLER
================================================================================

F5 Telemetry Streaming is a power tool that can be used to collect logs for for the BIG-IP System, and the applications it hosts.

In this module, you will:

-  Learn about F5 Telemetry Streaming (TS)
-  Verify Telemetry Streaming (TS) Is Installed on BIG-IP1 and BIG-IP2
-  Reveiw and Deploy a Telemetry Streaming (TS) Configuration to BIG-IP1 and BIG-IP2
-  Observe the different typs of Telemetry Streaming (TS) logs in AWS CloudWatch
