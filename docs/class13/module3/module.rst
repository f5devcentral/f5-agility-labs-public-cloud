App Deployment with Application Services 3 Extension (AS3)
================================================================================

Application Services 3 (AS3) is an F5 Automation Toolchain (ATC) extension that enables declarative configuration of  virtual servers.

In this module, you will:

-  Learn about the F5 extensions available for VS Code.
-  Install VS Code F5 extensions to manage F5 ATC components.
-  Deploy applications with AS3.
-  Convert an existing configuration into an AS3 Declaration.
