Deploying F5 Solutions to Google Cloud with Terraform and The F5 Automation Toolchain
============================================================================

This is a proctor led intermediate level lab.

In this lab, we'll demystify how Terraform integrates with the F5 Automation
and Orchestration toolchain to deploy to a shared GCP environment. Topics
covered include:

- Terraform
- F5 Cloud-init
- F5 Declarative Onboarding (DO)
- F5 Application Services 3 Extension (AS3)
- F5 Cloud Failover Extension (CFE)
- F5 Telemetry Streaming (TS)
